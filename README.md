# Data-Structure-OJ

这是 中国科学院大学 (UCAS) 数据结构 课程的 Online Judge 参考答案.

本仓库中的代码已经过2025春年度的OJ平台正确性验证, 因此大概率在之后也能通过.

## 文件说明

所有题目位于 `questions` 文件夹下, 子文件夹的命名格式为 `题号-ID-题目名称-Accepted`.

文件夹下 `main.c` 即为源码文件, `question.json` 为从平台手动爬取并生成的题目详情.
